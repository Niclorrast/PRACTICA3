# Mi App con Nginx + PHP-FPM + MySQL en Docker

## Descripción
Aplicación PHP desplegada en contenedores usando Docker Compose.

## Servicios
- Nginx (servidor web, puerto 8080)
- PHP-FPM (ejecutor de scripts PHP)
- MySQL (base de datos)
